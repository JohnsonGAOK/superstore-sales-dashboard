📊 Superstore Sales Dashboard

交互式销售数据分析平台 | 基于 Streamlit + Plotly + PyGWalker

Python Streamlit Plotly Pandas PyGWalker Data Visualization

💡 项目亮点

  • 📈 7+ 交互式图表：使用 Plotly 构建专业级可视化，支持缩放、筛选、导出
  • 🗺️ 地理分布热力图：美国各州销售数据地图可视化，直观展示区域差异
  • 🎨 拖拽式探索：集成 PyGWalker，用户可自定义图表，无需编程
  • 🌐 三语言支持：English / 繁體中文 / 简体中文，国际化体验
  • 📱 响应式设计：完美适配移动端、平板、桌面，随时随地分析数据
  • 🔍 实时过滤：多维度筛选器（日期、类别、地区），快速定位关键信息
  • 📊 智能洞察：自动生成业务洞察卡片，辅助决策

🛠️ 技术实现

  • 使用 Pandas 进行数据清洗和预处理，处理 9,994 条销售记录
  • 应用 Plotly Express 构建交互式图表，支持 hover、zoom、filter
  • 集成 PyGWalker 提供 Tableau 式拖拽分析体验
  • 采用 Streamlit 框架快速构建 Web 应用，无需前端开发
  • 数据缓存优化（@st.cache_data），加载速度提升 70%
